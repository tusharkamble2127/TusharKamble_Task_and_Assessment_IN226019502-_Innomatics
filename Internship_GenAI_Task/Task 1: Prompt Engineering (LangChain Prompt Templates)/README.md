# 🚀 GenAI Prompt Engineering using LangChain

This project is part of my **GenAI Internship**, where I built a **Mini Prompt Engine** using LangChain to generate dynamic and reusable prompts.

---

## 📌 Project Objective

The main goal of this assignment is to replace hardcoded prompts with **dynamic PromptTemplates** using LangChain and build a modular prompt generation system.

---

## ⚙️ Features Implemented

✔️ Dynamic PromptTemplate system (No hardcoding)
✔️ Multi-input prompt handling (topic, audience, tone)
✔️ Prompt variations engine (teaching, interview, storytelling)
✔️ ChatPromptTemplate system with roles
✔️ Input validation layer
✔️ Reusable prompt generator function
✔️ Template reusability testing

---

## 🧠 Key Learnings

* Designing reusable prompt systems instead of static prompts
* Importance of structured prompting in real-world GenAI applications
* Understanding LangChain PromptTemplate and ChatPromptTemplate
* Handling multiple inputs and validation

---

## 🔄 Workflow

User Input → Validation → Prompt Template → Dynamic Prompt Generation → Output
